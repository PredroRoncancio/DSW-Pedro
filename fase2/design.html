Propósito:

El Propósito de este sitio es fomentar el aprendizaje de diseño de sitios web,para tal motivo nuestra empresa HtmlDesign de Diseño de Sitios Web presenta uno de los recursos educativos digitales como son Etiquetas Básicas de HTML,atributos y valores para dinamizar las estrategias de aprendizaje.
Para el desarrollo del proyecto presenta al cliente un boceto el cual es el mockup de la página principal del sitio Web, si el cliente aprueba ese mockup se procede con la construcción del sitio web.
